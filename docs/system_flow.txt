DETAILED COMPONENT INTERACTIONS
================================

1. USER INPUT PHASE
   ┌─────────────┐
   │ User runs:  │
   │ python main.py │
   │ --role "SWE" │
   │ --resume doc │
   └─────────────┘
           │
           ▼

2. ORCHESTRATION PHASE
   ┌─────────────────────────────────┐
   │ Orchestrator Agent              │
   │ ├── Validates input             │
   │ ├── Creates session             │
   │ ├── Plans workflow              │
   │ └── Coordinates agents          │
   └─────────────────────────────────┘
           │
           ▼

3. JOB DISCOVERY PHASE
   ┌─────────────────────────────────┐
   │ Job Search Agent                │
   │ ├── Searches Indeed             │
   │ ├── Searches LinkedIn           │
   │ ├── Searches Glassdoor         │
   │ └── Returns job listings       │
   └─────────────────────────────────┘
           │
           ▼

4. SKILLS ANALYSIS PHASE
   ┌─────────────────────────────────┐
   │ Skills Analysis Agent           │
   │ ├── Extracts job descriptions   │
   │ ├── Uses OpenAI for analysis   │
   │ ├── Identifies required skills │
   │ └── Calculates skill priorities│
   └─────────────────────────────────┘
           │
           ▼

5. RESUME PROCESSING PHASE
   ┌─────────────────────────────────┐
   │ Resume Analysis Agent           │

   │ └── Identifies skill gaps      │
   └─────────────────────────────────┘

   ┌─────────────────────────────────┐

   │ ├── Uses OpenAI for optimization│

   │ └── Generates new resume       │
   └─────────────────────────────────┘
           │
           ▼

6. APPLICATION PHASE
   ┌─────────────────────────────────┐
   │ Application Agent               │
   │ ├── Navigates to job site      │

   │ ├── Uploads optimized resume   │
   │ └── Submits application        │
   └─────────────────────────────────┘
           │
