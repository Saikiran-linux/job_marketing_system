DATA FLOW DIAGRAM
=================

User Input → CLI/API → Orchestrator Agent
     ↓
Workflow Planning & State Management
     ↓
┌─────────────────────────────────────────────────────────────────┐
│                    EXECUTION FLOW                              │
├─────────────────────────────────────────────────────────────────┤
│  1. Job Search Agent → Multiple Job Sources                   │
│     ↓                                                          │
│  2. Skills Analysis Agent → OpenAI API → Skill Extraction     │
│     ↓                                                          │
│  3. Resume Analysis Agent → File System → Content Analysis    │
│     ↓                                                          │
│  4. Resume Modification Agent → OpenAI API → Optimization     │
│     ↓                                                          │
│  5. Application Agent → Web Automation → Job Submission       │
│     ↓                                                          │
│  6. Database Storage → Application Tracking                    │
│     ↓                                                          │
│  7. Report Generation → Analytics & Insights                   │
└─────────────────────────────────────────────────────────────────┘
